# 2526-POO-TEJADA_FROILAN
Este repositorio contiene ejemplos prácticos de las principales técnicas de Programación Orientada a Objetos (POO). 
Cada técnica tiene su propia carpeta con un archivo que incluye explicaciones y comentarios detallados para facilitar el aprendizaje. Este proyecto tiene como objetivo ayudar a comprender mejor la estructura y los beneficios de la POO para crear código modular, reutilizable y fácil de mantener.
